# Certificate Logo Assets

Place the final certificate and education logos in this folder.

The current `about.html` file expects these files:

| Card | Expected file | Notes |
| --- | --- | --- |
| Ege University - Computer Programming | `ege-university-logo.svg` | Replace with the official Ege University logo. |
| Data Analysis School - Artificial Intelligence Module | `data-analysis-ai-logo.svg` | Replace with the logo you want to use for the AI module. |
| TUSAS LIFT UP | `tusas-lift-up-logo.svg` | Replace with the TUSAS/LIFT UP logo you want to use. |

Recommended logo format:

- SVG is preferred for sharp rendering.
- PNG is fine if the logo is high resolution and has a transparent background.
- Keep each logo visually centered and readable inside a 96px square frame.
- If you switch to PNG, update the matching `src` values in `about.html`.
